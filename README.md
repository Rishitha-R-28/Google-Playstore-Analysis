# Google-Playstore-Analysis
Dataset containing Google Playstore app details
# 📊 Google Playstore Dataset
 
It includes information about mobile applications available on the Google Playstore, such as categories, ratings, reviews, size, installs, and more.

---

# File Contents
- * GOOGLE_PLAYSTORE_DATASET.csv *  
  The main dataset file containing app details.

---

# Dataset Features
Some of the key columns in the dataset:
- App – Name of the application
- Category – Type of app (e.g., Game, Productivity, Tools)
- Rating – Average user rating
- Reviews – Number of user reviews
- Size – App size
- Installs – Number of installs
- Type – Free or Paid
- Price – Price if paid
- Content Rating – Age group suitability
- Genres– App genres
- Last Updated – Date of last update
- Current Version – App version
- Android Version – Minimum required Android version

---

# Usage
This dataset can be used for:
- Data analysis and visualization
- Machine learning projects
- Market trend analysis
- App rating prediction models
